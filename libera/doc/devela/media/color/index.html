Chromatic functionality.

§Notes

  • Alpha channel is always linear, even in non-linear color spaces.

§Naming

  • Some prefixes specify color space (Rgb*: gamma-corrected, Lrgb*: linear).
  • Some suffixes specify component type and precision (*U8, *F32).
  • Other suffixes: Pre for premultiplied alpha.
