Struct devela::all::GraphU32

source · 
pub struct GraphU32<const VCAP: usize, const ECAP: usize, V = (), E = (), S: Storage = Bare> { /* private fields */ }
Expand description

A static graph with NonExtremeU32 indices, and two Arrays for vertices and edges.

It is generic in respect to the

  • vertices const capacity (VCAP),
  • edges const capacity (ECAP)
  • vertices data type (V),
  • edges data type (E),
  • storage (S).

Edges are represented with the EdgeU32s type.

§Features

It supports multiple index sizes by enabling the corresponding features: _graph_u[8|16|32|size].
