Expand description

Captures the first token from a list of inputs.

Usage: capture_first!(<category> <first>, <tail>*, <optional_comma>?);

§Example

assert_eq![5, capture_first!(expr 5, 6, 7)];
assert_eq![42, capture_first!(literal 42, "hello", true)];
