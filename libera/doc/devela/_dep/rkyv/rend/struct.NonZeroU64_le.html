Struct NonZeroU64_le

#[repr(C, align(8))]
pub struct NonZeroU64_le(/* private fields */);
Available on crate feature dep_rkyv only.
Expand description

A little-endian NonZeroU64 with a guaranteed size and alignment of 8.
