Available on crate feature dep_rkyv only.
Expand description

A variant of Serialize for “with” types.

See ArchiveWith for more details.

Required Methods§

fn serialize_with( field: &F, serializer: &mut S, ) -> Result<Self::Resolver, <S as Fallible>::Error> ⓘ

Serializes the field type F using the given serializer.

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
