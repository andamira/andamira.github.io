Available on crate feature std, or crate features linux and unsafe_syscall only.
Expand description

Prints to the standard error.

Leverages Linux::eprint , format_buf and join.

Usage is similar but not equal to std::eprint!.

ยงExamples

use devela::{os_eprint};

os_eprint!(); // prints nothing (1st arm)
os_eprint!("hello world! 2nd arm"); // one literal
os_eprint!("hello", " world! ", "3", "rd arm"); // many literals

os_eprint!(buf_a=30); // create a buffer of some byte lenght (4th arm)
os_eprint!(buf_a, "hello world! {}th arm", 5); // formatted print using the buffer
os_eprint!(buf_b=20, "hello world! {}th arm", 6);// create a buffer and use it to print

os_eprint!(&format!["{} {}! {}th arm", "hello", "world", 7]); // one &str expresion

Output:

hello world! 2nd arm
hello world! 3rd arm
hello world! 5th arm
hello world! 6th arm
hello world! 7th arm
