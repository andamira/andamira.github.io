Struct Pinned

pub struct Pinned<A, B = (), C = (), D = (), E = (), F = (), G = (), H = ()> { /* private fields */ }
Available on crate feature unsafe_ptr only.
Expand description

A wrapper for structurally pinned data.

Up to 8 generics can be supplied for 8 structurally pinned fields.
