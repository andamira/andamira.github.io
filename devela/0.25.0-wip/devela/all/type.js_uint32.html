⚙️ 𝟙 A JavaScript unsigned 32-bit integer.

JavaScript lacks native unsigned integers, but the >>> operator treats numbers as unsigned u32. Some APIs, such as Uint32Array, also provide unsigned integer behavior.

  • To ensure unsigned 32-bit behavior in JavaScript: use Uint32Array instead of plain arrays.
  • To coerce a number into a u32: use num >>> 0 in JavaScript.
  • Only >>> (unsigned right shift) preserves unsigned semantics.
