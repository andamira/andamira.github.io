1// devela::num::niche
2//
3//! Specialized numeric types and behaviors.
4//!
5//! Prohibit specific values with zero-cost optimizations,
6//! enabling memory-efficient data structures and domain modeling.
7//!
8//! ## Core Types
9//!
10//! - `NonZero[I|U]*` (re-exported)
11//!   - Standard zero-prohibiting types with niche optimization.
12//!
13//! - `NonValue[I|U]*<const V>`.
14//!   - General extension of `NonZero*` guaranteeing `value != V`.
15//!   - **Implementation**: Stores transformed value in `NonZero*`.
16//!   - **Optimizations**: Automatic instruction selection per case.
17//!
18//! ## Recommended Defaults
19//!
20//! ### `NonExtremeU*` = `NonValueU*<MAX>`
21//! - Need to preserve zero but prohibit MAX.
22//! - Working with indices/counters where MAX is special.
23//! - Ideal for: collection indices, counters, bitmask handling.
24//! - **Optimization**: Single `NOT` instruction.
25//!
26//! ### `NonExtremeI*` = `NonValueI*<MIN>`
27//! - Need zero but want symmetric range.
28//! - Mathematical contexts where MIN is problematic.
29//! - Ideal for: mathematical ranges, circular buffers, DSP algorithms.
30//! - **Optimization**: `LEA` instruction fusion.
31//!
32//! ## Optimization Characteristics
33//! | Type                | Prohibits | Storage       | Optimization | vs `NonZero*`          |
34//! |---------------------|-----------|---------------|--------------|------------------------|
35//! | `NonExtremeU*`      | MAX       | `!value`      | `NOT`        | Keeps zero, drops MAX  |
36//! | `NonExtremeI*`      | MIN       | `value ^ MIN` | `LEA`        | Keeps zero, drops MIN  |
37//! | `NonValue*`         | Custom V  | `value ^ V`   | `XOR`        | Fully general          |
38//! | `NonZero*`          | 0         | raw value     | -            | Classic case           |
39//!
40//! ## Usage Guide
41//! | Use Case                  | Recommended Type          | Advantage                         |
42//! |---------------------------|---------------------------|-----------------------------------|
43//! | Must prohibit zero        | `NonZero*`                | Standard solution                 |
44//! | Need sentinel values      | `NonValue*<SENTINEL>`     | Custom prohibited value           |
45//! | Index/counter handling    | `NonExtremeU*`            | Avoids overflow edge cases        |
46//! | Mathematical purity       | `NonExtremeI*`            | Symmetric range                   |
47//! | Maximum flexibility       | Primitive types           | No constraints                    |
48//
