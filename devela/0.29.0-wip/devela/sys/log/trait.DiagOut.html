Expand description

🪵 Emits leveled diagnostic text.

📍 sys/log

This is the minimal semantic sink for diagnostics. It layers above plain text output by attaching a DiagLevel to each emitted message.

See also TextOut for non-leveled textual output.

impl DiagOut for JsConsole

Available on crate features js and unsafe_ffi only.

Emits leveled diagnostics to the JavaScript console.

This implementation maps DiagOut directly onto the browser’s native console levels, preserving the semantic distinction between trace, debug, info, warning, and error output.
