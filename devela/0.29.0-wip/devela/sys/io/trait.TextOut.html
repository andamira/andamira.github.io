🔌 𝐓 Emits UTF-8 text to an output surface.

📍 sys/io

This is the text-layer counterpart to IoWrite. It is intended for sinks whose primary semantic payload is text, such as terminals, log files, browser text surfaces, or formatted buffers.

See also IoWrite for byte-oriented output.
