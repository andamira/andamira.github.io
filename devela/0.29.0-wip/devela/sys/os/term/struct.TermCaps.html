Struct TermCaps

pub struct TermCaps { /* private fields */ }
Available on crate feature term only.
Expand description

🮖 ⬡ ⊆ ▫️ Terminal capability bits.

📍 sys/os/term

📦 size_of::<TermCaps>() == 4 bytes / 32 bits

Stores independent terminal feature flags plus the maximum known color depth.

See also TermCap.
