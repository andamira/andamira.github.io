Struct LinuxSignalSet 

Source 
pub struct LinuxSignalSet { /* private fields */ }
Available on crate feature linux only.
Expand description

🐧 📶 ⊆ A compact semantic set of standard Linux signals.

📍 sys/os/linux/process

📦 size_of::<LinuxSignalSet>() == 4 bytes / 32 bits

This is not the raw Linux sigset_t mask. Signal bits are stored compactly from zero: SIGHUP is bit 0, SIGINT is bit 1, …
