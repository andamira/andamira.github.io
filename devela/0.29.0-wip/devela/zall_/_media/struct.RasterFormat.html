pub const fn bits_per_sample(self) -> Option<u16> ⓘ

Returns the number of bits used by one unpacked sample.

For packed formats, prefer bits_per_pixel, since channels are bit fields inside one scalar element.

Source

pub const fn depth_bits(self) -> Option<u16> ⓘ

Returns the number of meaningful bits per pixel.

Padding bits are excluded. Alpha bits are included when present.

Source

pub const fn bits_per_pixel(self) -> Option<u16> ⓘ

Returns the number of stored bits per pixel.

Padding bits are included.

Source

pub const fn stored_bytes_per_pixel(self) -> Option<u16> ⓘ

Returns the number of stored bytes per pixel, if byte-aligned.

Padding bits are included.
