Expand description

🔄 🍃 A lending iterator that can yield items from the back.

📍 data/access/iter

This is the borrowing analogue of : IteratorDoubleEnded.

Required Methods§

fn next_back<'a>(&'a mut self) -> Option<Self::Item<'a>> ⓘ

Returns the next item from the back of the iterator.

Provided Methods§

Source

fn nth_back<'a>(&'a mut self, n: usize) -> Option<Self::Item<'a>> ⓘ

Skips n items from the back and returns the next one.

Repeatedly calls next_back n + 1 times. Returns the final yielded item, or None if the iterator ends before advancing n steps.

This is the reverse analogue of IteratorLending::nth.

Source

fn try_rfold<B, E, F>(&mut self, acc: B, f: F) -> Result<B, E> ⓘ
where F: FnMut(B, Self::Item<'_>) -> Result<B, E>,

Reverse-direction equivalent of IteratorLending::try_fold.

Applies f starting from the back of the iterator, folding items in reverse order. Stops on first Err.
