𝐓 ::= A byte scanner over source text.

📍 text/parse

TextScanner provides incremental, allocation-free traversal over a borrowed text source, exposing byte-oriented operations suitable for building parsers.

Source

pub const fn next_line_trimmed_before(&mut self, byte: u8) -> Option<TextRange> ⓘ

Consumes and returns the next line before byte, trimmed of ASCII horizontal whitespace.

The delimiter byte is not consumed as part of the returned range, but the full source line and its line ending are consumed.

This is useful for simple line-comment formats, for example with b'#'. It is byte-blind and does not understand quoted strings.

Returns None at end of input.

pub const fn take_quoted_basic( &mut self, ) -> Result<Option<TextRange>, TextParseError> ⓘ

Consumes a double-quoted string and returns the inner byte range.

The returned range excludes the surrounding quotes.

Recognized escapes: \", \\, \n, \r, \t, \0

This method validates escape syntax but does not decode escapes.

Returns Ok(None) if the next byte is not " .

Source

pub const fn take_quoted_basic_or_rest( &mut self, ) -> Result<TextRange, TextParseError> ⓘ

Consumes a basic double-quoted string, or the remaining unread input.

If the next byte is ", returns the inner quoted range, validating basic escapes as take_quoted_basic does.

Otherwise consumes and returns the rest of the input.

Source

pub const fn take_quoted_literal( &mut self, ) -> Result<Option<TextRange>, TextParseError> ⓘ

Consumes a single-quoted literal string and returns the inner byte range.

The returned range excludes the surrounding quotes.

Backslashes have no special meaning here.

Returns Ok(None) if the next byte is not '.

Source

pub const fn decode_quoted_basic_into( &self, range: TextRange, out: &mut [u8], ) -> Result<usize, TextParseError> ⓘ

Decodes a basic quoted string payload into out.

The input range is expected to be the inner range returned by take_quoted_basic, excluding the surrounding quotes.

Recognized escapes: \", \\, \n, \r, \t, \0

Returns the number of bytes written into out.

§Errors

Returns:

  • InvalidEscape for an unknown escape
  • UnterminatedQuote if the range ends with a trailing backslash
  • BufferTooSmall if out is too small
Source

pub const fn decode_quoted_basic_str_into<'str>( &self, range: TextRange, out: &'str mut [u8], ) -> Result<&'str str, TextParseError> ⓘ

Decodes a basic quoted string payload into out and returns it as &str.

This is a convenience wrapper over decode_quoted_basic_into.

§Errors

Returns the same decoding errors as decode_quoted_basic_into. Returns InvalidEscape as a fallback if the decoded bytes are not valid UTF-8.
