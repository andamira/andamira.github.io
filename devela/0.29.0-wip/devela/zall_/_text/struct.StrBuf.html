Struct StrBuf

pub struct StrBuf<'a> { /* private fields */ }
Expand description

🧶 A borrowed UTF-8 string buffer over caller-provided bytes.

📍 text/str

It stores a valid UTF-8 prefix inside a mutable byte slice, leaving the remaining bytes as spare capacity.

This is the borrowed counterpart to fixed-capacity owned strings.

pub fn format( buf: &'a mut [u8], args: FmtArguments<'_>, ) -> Result<&'a str, &'a str> ⓘ

Writes formatted output into buf.

Returns:

  • Ok(&str) if all formatting writes completed.
  • Err(&str) with the valid partial result if formatting stopped early.

Unlike FmtWriter, this uses StrBuf’s FmtWrite implementation: every write_str chunk is all-or-nothing.
