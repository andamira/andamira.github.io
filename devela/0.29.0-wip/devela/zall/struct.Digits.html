Struct Digits 

Source 
#[repr(transparent)]
pub struct Digits<T: Copy>(pub T);
Expand description

𝐓 ⅀ 🛠️ Provides ASCII digit operations and conversions for unsigned integer primitives.

📍 text

Enables efficient ASCII digit extraction, counting, and conversion for unsigned integer types. All operations are const and designed for performance-critical scenarios like number formatting.

It converts numbers → digits for display/formatting.

§Methods

Most methods are implemented for all the unsigned integer primitives: u8, u16, u32, u64, u128, usize.

Common methods, u8 version links:

  • count_digits10, count_digits16.
  • digit_at_index10 (checked), digit_at_index16 (checked).
  • digit_value_at_index10 (checked),
  • digit_value_at_index16 (checked),
  • digits10, digits16.
  • write_digits10 (nonzero).
  • write_digits10_fast (nonzero).
  • digits10_str, digits16_str.

Exclusive for u8:

  • digits10_1,
  • digits10_2, digits16_1.
  • write_digits16 (nonzero).

Exclusive for u16:

  • digits10_2, digits10_3, digits10_4.

§Examples

let dec = Digits(12345_u32);
assert_eq!(dec.digit_at_index10(0), b'5');
assert_eq!(dec.digit_at_index10(4), b'1');
assert_eq!(dec.digit_value_at_index10(2), 3);
assert_eq!(dec.count_digits10(), 5);
assert_eq!(dec.digits10(), *b"0000012345");
assert_eq!(Slice::trim_leading(&dec.digits10(), b'0'), b"12345");

let hex = Digits(0x89ABC_u32);
assert_eq!(hex.digit_at_index16(1), b'B');
assert_eq!(hex.digit_value_at_index16(1), 11);
assert_eq!(hex.count_digits16(), 5);
assert_eq!(hex.digits16(), *b"00089ABC");
assert_eq!(Slice::trim_leading(&hex.digits16(), b'0'), b"89ABC");

pub const fn write_digits10(self, buf: &mut [u8], offset: usize) -> usize

Writes 1..=3 decimal digits without leading zeros starting at offset, returning the number of bytes written.

Writes 0 when the value is zero.

This method checks the exact number of bytes required before writing. It can write into buffers with fewer than 3 bytes remaining when the value itself needs fewer digits. Returns 0 and writes nothing if insufficient space remains.
