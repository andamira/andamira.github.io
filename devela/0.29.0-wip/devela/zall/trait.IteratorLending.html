Expand description

🔄 🍃 A lending iterator using a generic associated lifetime.

📍 data/access/iter

A lending iterator yields items borrowed from its own internal state. Each call to next creates a temporary borrow of self and returns an item tied to that borrow. This enables iteration over memory-backed structures without copying or owning the underlying data.

This is the borrowing analogue of Iterator.

See also:

  • IteratorLendingDoubleEnded
  • IteratorLendingExactSize
  • IteratorLendingPeek
  • IteratorLendingPeekDoubleEnded

Required Associated Types§

Source

type Item<'a> where Self: 'a

The item yielded by this iterator.

The lifetime 'a corresponds to the borrow of self required to produce the item.

Required Methods§

Source

fn next<'a>(&'a mut self) -> Option<Self::Item<'a>> ⓘ

Advances the iterator and returns the next item.

The lifetime of the returned reference is scoped to the borrow of self established during this call.

Provided Methods§

Source

fn size_hint(&self) -> (usize, Option<usize>) ⓘ

Returns a size hint for the remaining items.

The default implementation returns (0, None), which is correct but conservative. Implementors may override this for improved accuracy.

Source

fn count(&mut self) -> usize

Consumes the iterator, returning the number of remaining items.

Source

fn advance_by(&mut self, n: usize) -> Result<(), usize> ⓘ

Advances the iterator by n steps.

Returns Ok(()) if n items were successfully skipped. Returns Err(remaining) if the iterator ended early, where remaining is the number of items that were not skipped.

Source

fn nth<'a>(&'a mut self, n: usize) -> Option<Self::Item<'a>> ⓘ

Advances the iterator by n items and returns the next one.

Source

fn for_each<F>(&mut self, f: F)
where F: FnMut(Self::Item<'_>),

Applies f to each remaining item.

Source

fn inspect<F>(&mut self, f: F)
where F: FnMut(&Self::Item<'_>),

Calls f for each item before yielding it.

This is useful for debugging or side effects. The iterator behavior is otherwise unchanged.

Source

fn is_partitioned<P>(&mut self, pred: P) -> bool
where P: FnMut(&Self::Item<'_>) -> bool,

Returns true if the iterator is partitioned according to pred.

All items for which pred returns true must come before all items for which it returns false. Once the predicate becomes false, it must remain false for all subsequent items.

Source

fn try_fold<B, E, F>(&mut self, acc: B, f: F) -> Result<B, E> ⓘ
where F: FnMut(B, Self::Item<'_>) -> Result<B, E>,

Applies f to each item, accumulating results in acc.

Iteration stops on the first Err. Returns Ok(acc) when all applications succeed.

This mirrors Iterator::try_fold but works with borrowed items.

Source

fn try_for_each<F, E>(&mut self, f: F) -> Result<(), E> ⓘ
where F: FnMut(Self::Item<'_>) -> Result<(), E>,

Applies f to each remaining item until an error occurs.

Source

fn all<P>(&mut self, pred: P) -> bool
where P: FnMut(&Self::Item<'_>) -> bool,

Returns true if all items satisfy the predicate.

Source

fn any<P>(&mut self, pred: P) -> bool
where P: FnMut(&Self::Item<'_>) -> bool,

Returns true if any item satisfies the predicate.

Source

fn position<P>(&mut self, pred: P) -> Option<usize> ⓘ
where P: FnMut(&Self::Item<'_>) -> bool,

Returns the index of the first item for which pred returns true, or None if no matching item is found.
