Struct Order

pub struct Order;
Expand description

⅀ 🧱 🛠️ Unified namespace for multi-dimensional ordering schemes.

📍 num/fin/ord

Order provides a set of static functions that map discrete coordinates from higher-dimensional integer spaces to a single linear index, and back, using different locality-preserving strategies.

pub const fn grid_volume<const N: usize>(dims: [usize; N]) -> Option<usize> ⓘ

Returns the number of coordinates in the grid.

Returns None if the product overflows usize.

An empty dimension makes the grid empty. A zero-dimensional grid has one coordinate, following the empty-product convention.

Source

pub const fn row_major_strides<const N: usize>( dims: [usize; N], ) -> Option<[usize; N]> ⓘ

Computes row-major strides.

The returned first stride is 1.

For [width, height, depth], this returns [1, width, width * height].

Source

pub const fn col_major_strides<const N: usize>( dims: [usize; N], ) -> Option<[usize; N]> ⓘ

Computes column-major strides.

The returned last stride is 1.

For [width, height, depth], this returns [height * depth, depth, 1].
