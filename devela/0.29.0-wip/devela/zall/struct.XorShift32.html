🎲 The XorShift32 PRNG.

📍 num/prob/rand

It has a 32-bit state and generates 32-bit numbers.

This is the classic 32-bit XorShift algorithm by George Marsaglia.

The BASIS and triplet (A, B, C) values default to the canonical example.
