Expand description

🎲 🏗️ Fallible construction from a source of randomness.

📍 num/prob/rand

FromRandTry defines a type’s canonical random construction.

Implementations should document their distribution and constraints. Types without a clear canonical distribution should use explicit constructors or distribution types instead.
