Expand description

🌱 🗃️ Initializes a [$T; $LEN] array in multiple ways.

📍 data/layout/array

§Arguments

  • [$T; $LEN]: the array’s elements’ type and length.
  • $init: a function with an usize argument that returns $T.
  • $const_fn: a const fn with an usize argument that returns $T: Copy.
  • $copiable: an expression that returns an element of type $T: Copy.
  • $clonable: an expression that returns an element of type $T: Clone.
  • $fsafe: a feature that forbids the use of unsafe when enabled.
  • $funsafe: a feature that enables the use of unsafe when enabled.
  • $intoiter: an item that implements IntoIterator.
  • $trait: in the INIT in arm, the path to either ConstInit or ConstInit.

§Examples

assert_eq![[2,4,6], init_array![safe_init [i32; 3], |n| (n as i32 + 1) * 2]];
#[cfg(feature = "unsafe_array")]
assert_eq![[3,6,9], init_array![unsafe_init [i32; 3], |n| (n as i32 + 1) * 3]];

assert_eq![[2,4,6], init_array![init [i32; 3], "safe", "unsafe_array",
    |n| (n as i32 + 1) * 2]];
assert_eq![Box::new([2,4,6]), init_array![init_heap [i32; 3], "safe", "unsafe_array",
    |n| (n as i32 + 1) * 2]];

assert_eq![[7,7,7], init_array![clone [i32; 3], "safe", "unsafe_array", 7]];
assert_eq![[0,0,0], init_array![default [i32; 3], "safe", "unsafe_array"]];
assert_eq![[4,5,6], init_array![iter [i32; 3], "safe", "unsafe_array", vec![4,5,6,7,8]]];
assert_eq![[4,0,0], init_array![iter [i32; 3], "safe", "unsafe_array", vec![4]]];

const fn init(n: usize) -> i32 { (n as i32 + 1) * 4 }
const ARRAY1: [i32; 3] = init_array![const_fn [i32; 3], "safe", "unsafe_array", init, 0];
assert_eq![[4, 8, 12], ARRAY1];

const ARRAY2: [i32; 3] = init_array![INIT in ConstInit [i32; 3]];
assert_eq![[0, 0, 0], ARRAY2];

§Features

The unsafe version uses MaybeUninit in the case of stack allocation or Box::from_raw in the case of heap allocation.

For the const, clone, default and iter versions, if the given $funsafe is enabled and the given $fsafe is disabled, it will use unsafe initialization.

§Notes

For the heap-related arms needs to have Vec in scope.
