6#[doc = crate::_tags!(num namespace)]
7/// Provides constant bitwise operations on `T`.
8#[doc = crate::_doc_meta!{location("num/fin")}]
9///
10/// It's implemented for: [`u8`], [`u16`], [`u32`], [`u64`], [`u128`] and [`usize`].
11///
12/// ## Panic behavior
13/// Unchecked bit operations panic in debug builds when a bit index or range
14/// is out of bounds. In release builds they do not panic; the invalid
15/// index produces a wrapped shift and a non-meaningful result.
16///
17/// Checked variants never panic and return an error instead.
18///
19/// # Related items
20/// See also [`BitOps`][crate::BitOps] for the related trait.
21///
22/// # Methods
23///
24/// The methods are the same for all unsigned primitives.
25/// The following list of methods links to the `u8` implementation:
26///
27/// - [Constants and mask constructors](#constants-and-mask-constructors-for-u8)
28///   - [mask_range](#method.mask_range) ([*checked*](#method.mask_range_checked)).
29///   - [is_set_mask](#method.is_set_mask).
30///   - [set_mask](#method.set_mask).
31///   - [is_unset_mask](#method.is_unset_mask).
32///   - [unset_mask](#method.unset_mask).
33///
34/// - [Get methods](#get-methods-for-u8)
35///   - [get_range](#method.get_range) ([*checked*](#method.get_range_checked)).
36///   - [get_value_range](#method.get_value_range) ([*checked*](#method.get_value_range_checked)).
37///
38/// - [Set ops](#set-ops-for-u8)
39///   - [is_set](#method.is_set) ([*checked*](#method.is_set_checked)).
40///   - [set](#method.set) ([*checked*](#method.set_checked)).
41///   - [is_set_range](#method.is_set_range) ([*checked*](#method.is_set_range_checked)).
42///   - [set_range](#method.set_range) ([*checked*](#method.set_range_checked)).
43///   - [set_all](#method.set_all).
44///   - [set_value_range](#method.set_value_range)
45///     ([*checked*](#method.set_value_range_checked),
46///     [*checked_strict*](#method.set_value_range_checked_strict)).
47///
48/// - [Unset ops](#unset-ops-for-u8)
49///   - [is_unset](#method.is_unset) ([*checked*](#method.is_unset_checked)).
50///   - [unset](#method.unset) ([*checked*](#method.unset_checked)).
51///   - [is_unset_range](#method.is_unset_range) ([*checked*](#method.is_unset_range_checked)).
52///   - [unset_range](#method.unset_range) ([*checked*](#method.unset_range_checked)).
53///   - [unset_all](#method.unset_all).
54///
55/// - [Flip ops](#flip-ops-for-u8)
56///   - [flip](#method.flip) ([*checked*](#method.flip_checked)).
57///   - [flip_range](#method.flip_range) ([*checked*](#method.flip_range_checked)).
58///   - [flip_range_if](#method.flip_range_if) ([*checked*](#method.flip_range_if_checked)).
59///
60/// - [Reverse ops](#reverse-ops-for-u8)
61///   - [reverse_range](#method.reverse_range) ([*checked*](#method.reverse_range_checked)).
62///
63/// - [Count ops](#count-ops-for-u8)
64///   - [count_ones_range](#method.count_ones_range)
65///     ([*checked*](#method.count_ones_range_checked)).
66///   - [count_zeros_range](#method.count_zeros_range)
67///     ([*checked*](#method.count_zeros_range_checked)).
68///
69/// - [Find ops](#find-ops-for-u8)
70///   - [find_first_one_range](#method.find_first_one_range)
71///     ([*checked*](#method.find_first_one_range_checked)).
72///   - [find_first_zero_range](#method.find_first_zero_range)
73///     ([*checked*](#method.find_first_zero_range_checked)).
74///   - [find_last_one_range](#method.find_last_one_range)
75///     ([*checked*](#method.find_last_one_range_checked)).
76///   - [find_last_zero_range](#method.find_last_zero_range)
