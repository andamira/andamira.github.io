19/// Unicode scalars-related low-level *const* operations.
20#[doc = crate::_doc_meta!{location("text/unicode/scalar")}]
21///
22/// # Terminology
23/// - A **code point** is an integer in `U+0000..=U+10FFFF`.
24/// - A **surrogate** is a code point in `U+D800..=U+DFFF`.
25/// - A **Unicode scalar value** is a code point that is not a surrogate.
26///   Rust's [`char`][prim@char] represents exactly this set.
27/// - A **scalar rank** is the zero-based position of a scalar value in the
28///   ordered set of all Unicode scalar values, with the surrogate range omitted.
29///
